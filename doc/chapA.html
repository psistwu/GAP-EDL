A Utilities

Here, we list utilities which are used throughout the package.

A.1 EquiDeg Info Class

A.1-1 INFO_LEVEL_EquiDeg
‣ INFO_LEVEL_EquiDeg( global variable )

is the info level regarding the usage of Info (Reference: Info) throughout the package; its default value is 1. Since higher value results in more detailed feedback, one can increase the value for debugging purpose.

A.1-2 InfoEquiDeg
‣ InfoEquiDeg( info class )

is the default info class throughout the package. Its default info level is 1.

A.2 LaTeX Typesettings

A.2-1 LaTeXString
‣ LaTeXString( obj )( attribute )

stores the LaTeX representation of an object. This attribute is supposed to be assigned manually.

A.2-2 LaTeXTypesetting
‣ LaTeXTypesetting( obj )( attribute )

returns the LaTeX typesetting of an object. In default, it simply calls LaTeXString (A.2-1). However, it may behave differently for objects of different categories.
